# WokaMeeting

A decentralised P2P meeting platform

https://bartoszstepien01.github.io/WokaMeeting
## Building
```bash
git clone https://github.com/bartoszstepien01/WokaMeeting.git
cd WokaMeeting
npm install
npm run build
```
## Developing
```bash
git clone https://github.com/bartoszstepien01/WokaMeeting.git
cd WokaMeeting
npm install
npm run dev
```
